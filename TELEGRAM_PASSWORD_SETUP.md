# 🔐 Настройка пароля при регистрации через Telegram

## 📋 Текущая проблема

При регистрации через Telegram автоматически генерируется случайный пароль, который пользователь не знает. Это создает проблемы:
- Пользователь не может войти через обычную форму
- Нет возможности восстановить доступ, если Telegram недоступен

## 🎯 Решения

### Вариант 1: Показать пароль после регистрации (РЕАЛИЗОВАНО) ⭐

После регистрации через Telegram показывается модальное окно с:
- Сгенерированным паролем
- Возможностью скопировать пароль
- Предупреждением сохранить пароль
- Кнопкой "Установить свой пароль"

### Вариант 2: Отправить пароль на email

Если у пользователя есть email (из Telegram или введен дополнительно), пароль отправляется на почту.

### Вариант 3: Запросить пароль при первой регистрации

После авторизации через Telegram показывается форма для установки пароля.

## ✅ Реализованное решение

1. **Показ пароля в модальном окне** после регистрации
2. **Возможность установить свой пароль** сразу после регистрации
3. **Отправка пароля на email** (если email доступен)
4. **Ссылка на страницу установки пароля** в профиле

## 🔧 Как это работает

1. Пользователь регистрируется через Telegram
2. Генерируется случайный пароль
3. Показывается модальное окно с паролем
4. Пользователь может:
   - Скопировать пароль
   - Установить свой пароль
   - Продолжить без установки (может установить позже в профиле)

## 📝 Настройка

Все настройки уже реализованы в коде. Никаких дополнительных действий не требуется.
